# Basic Network Sniffer

This is a simple Network Sniffer tool developed during my internship at *CodeAlpha*. The tool captures network packets and displays source/destination IP addresses and protocols.

## Features
* Captures real-time network traffic.
* Extracts IP layer information.
* Identifies network protocols.

## Requirements
* Python 3.x
* Scapy library (pip install scapy)
